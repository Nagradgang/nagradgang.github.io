Table Tags

Tag Name Description
<table> Table Defines a table
<caption> Caption Provides a title or caption for the table
<thead> Table Head Used to group header rows in a table
<tbody> Table body Shows the main body of the table
<tfoot> Table footer Creates a separate footer for the table
<tr> Table Row Defines a table row
<th> Table Header Cell Used to define header cells in a table
<td> Table Data Defines a standard cell in a table

CSS-selectors

Selector Example Description
element p Element Selector
Selects all <p> elements
#id #firstname ID Selector
Selects the element with id="firstname"
.class .intro Class Selector
Selects all elements with a specific class attribute
[attribute=value] [target="blank"] Attribute Selector
Selects elements based on their attributes
element element div p Descendant Selector
Selects all <p> elements inside <div> elements
element>element div > p Child Selector
Selects all <p> elements where the parent is a <div> element
h2 + p element+element Adjacent Sibling Selector
Selects the first <p> element that is placed immediately
after <div> elements
p ~ ul element1~element2 General Sibling Selector
Selects every <ul> element that is preceded by a <p> element
